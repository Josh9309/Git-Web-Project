Why I Chose This Topic

I chose to do my tutorial on GitHub because it is shocking how in the IGM GDD major not many students have a good grasp on what GitHub is or the basic for using it. It is very important in are industry to know how to properly use version control because you will come across case where your code will break and you will need to fall back to a previous version. Not to mention that GitHub is also a great Portfollio tool for code, because it allows employers to see your coding skill and your work ethic.
