Welcome to the World of GitHub

Hello and welcome to the wonderous world of GitHub!!!!

If you are at this page your are probably very new to GitHub and probably want to learn all you can about GitHub and how it works. Before we begin let me tell you a little bit about what GitHub is. GitHub is a version control software made using git version control coding lanquage. GitHub is one of many other places that use git, other places include bitBucket, source-tree and more. The first thing to understand about GitHub is that it is not Google drive, it is not used to back up code projects. It is used to mangage code changes in code projects. What that means is that when you are working on a project and finish big feature you want to make a commit and push that to git hub. It doesn't means that you want to work on code for an hour and commit push unfinished code especially if you are using GitHub in a Group.

So anyways in this tutorial I will show you how to enter the world of GitHub, how to make a repository and how to commit and synch with github.
