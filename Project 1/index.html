Step 1: Sign up and download GitHub

Hi, Today I am going to teach you how to use Github to manage your coding project's code. For the first step you will need to sign up and download github for windows or mac. The links are provided bellow.

  • GitHub account Sign up
  • GitHub Desktop Download
    Select Windows or Mac depending on your computer

Step 2: Create a repository

After you have download the desktop app and logged into it. you will need to create a repository for your project to hold the project files. On the left side of the desktop application you will see a "+" click the "+" and select the create tab. Choose a name for your project and select a Git ignore for what type of project you are doing.
The Git Ignotre tells GitHub which files to not worry about this is usefull when doing projects in visual studios or unity.

Step 3: Make a project in the repository folder

once you have made your repository, create your project to that folder. If you want for the sake of this tutorial you can just make a text file, add a couple lines of text and save it to the folder.
